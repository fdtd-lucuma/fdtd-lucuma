# Una GUI para fdtd

## Dependencies

* `bash-completion`
* `cmake`
* `ninja`

## Build
``` bash
git clone https://github.com/fdtd-lucuma/fdtd-vulkan
cd fdtd-vulkan

cmake -B build
cmake --build build

cd build
./fdtd-vulkan
```
